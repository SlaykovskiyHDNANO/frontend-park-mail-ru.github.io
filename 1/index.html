Приступаем!

            $ mkdir epicgame & cd epicgame
            $ npm init
            $ cat package.json
            $ npm install grunt-cli -g
            $ npm grunt grunt-contrib-connect --save-dev
            $ vim Gruntfile.js

Gruntfile.js

            module.exports = function (grunt) {
                grunt.initConfig({
                    connect: { … } /* Задача */
                });
                grunt.loadNpmTasks('grunt-contrib-connect');
            };

Задача connect

            server: { /* Подзадача */
                options: {
                    keepalive: true, /* работать постоянно */
                    port: 8080, /* номер порта */
                    base: 'www' /* публичная директория */
                }
            }
