ГЛАВНАЯ

Преамбула

Предполагается, что студент имеет базовые знания веб-технологий (HTML, CSS, JavaScript) и практические навыки решения задач с помощью базовых алгоритмов и структур данных.

Проектная работа выполняется в группах по 2–3 человека (роли каждого должны быть представлены при формировании группы).

Для выполнения проектной работы на занятиях потребуется ноутбук/виртуальная машина с установленной ОС Linux/MacOS/Windows. Из программного обеспечения — Git и Node.js 0.8+.

Ресурсы для самостоятельного изучения:

  1. Влад Мержевич. Самоучитель HTML4 // 2010.
  2. Влад Мержевич. Самоучитель CSS // 2010.
  3. Илья Кантор. Современный учебник JavaScript // 2013.
  4. Антон Шевчук. jQuery для начинающих // 2013.
  5. Mark Piligrim. Dive into HTML5 // 2013.

Интересные ресурсы:

  1. Mozilla Developer Network.
  2. HTML5 Rocks.
  3. Build New Games. Open Web techniques for cutting-edge game development.
  4. DevDocs.
