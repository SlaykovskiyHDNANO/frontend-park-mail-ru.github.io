Основные элементы формы

  1. Текстовые поля - input[type=text], textarea
  2. Радиокнопки - input[type=radio]
  3. Флажки или чекбоксы - input[type=checkbox]
  4. Выпадающие списки - select и options
  5. Кнопки - input[type=submit][type=reset], button[type=submit][type=reset]

Пример валидации

Проверка выполняется:

  • Во время ввода (oninput)
  • Перед вводом (onfocus)
  • По окончании ввода (onblur)
  • Перед отправкой формы (onsubmit)

Сообщение об ошибке

  • Выделение поля с ошибкой
  • Поясняющее текстовое сообщение об ошибке
  • Фокус на поле ввода с ошибкой
  • Блокировка кнопки для отправки формы
