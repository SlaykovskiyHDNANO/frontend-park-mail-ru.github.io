Установка

            Ruby (Windows)
            $ sudo gem install sass
            $ sass --version

Использование

            $ sass in.scss out.css
            $ sass --watch in.scss:out.css

grunt-contrib-sass

            $ npm install grunt-contrib-sass --save-dev

Gruntfile.js

            grunt.initConfig({
                sass: { … } /* grunt-contrib-sass */
            });
            
            grunt.loadNpmTasks('grunt-contrib-sass');

grunt-contrib-sass

            css: { /* Подзадача */
                files: [{
                    expand: true,
                    cwd: 'public/css', /* исходная директория */
                    src: '*.scss', /* имена шаблонов */
                    dest: 'public/css' /* результирующая директория */
                }]}

SCSS: Variables

            $font-family: 'PT Serif', serif;
            $font-size: 18px;
            $line-height: $font-size * 1.5;
            $color: #333;
            body {
                font: normal #{$font-size}/#{$line-height} $font-family;
                color: $color; }

CSS

            body {
                font: normal 18px/27px 'PT Serif', serif;
                color: #333; }

SCSS: Nesting

            ul {
                padding: 0;
                li {
                    display: block;
                    &:hover {
                        color: #eee; }
                }
            }

CSS

            ul {
                padding: 0; }
            ul li {
                display: block; }
            ul li:hover {
                color: #eee; }

SCSS: Import & Partials

            /* _reset.scss */
            html, body, ul, ol {
                margin: 0; padding: 0; }
        
            // main.scss
            @import "reset";
            body { color: #333; }

CSS

            html, body, ul, ol {
                margin: 0; padding: 0; }
            body {
                color: #333; }

SCSS: Mixins

            @mixin box-sizing($value) {
                -webkit-box-sizing: $value;
                -moz-box-sizing: $value;
                box-sizing: $value; }
            .button {
                color: #333;
                @include box-sizing(border-box); }

CSS

            .button {
                color: #333;
                -webkit-box-sizing: border-box;
                -moz-box-sizing: border-box;
                box-sizing: border-box; }

            .message {
